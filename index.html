Music Commands
Commands
play • Joins and play music in a channel.
pause • Pause or resume the music player.
stop • Stop and clear the music player.
skip • Skip the current music track if you're the requester.
remove • Remove a song from the queue.
move • Move the bot to a channel.
lyrics • Get the lyrics of the currently playing song, or others.
shuffle • Shuffle the music queue.
nowplaying • Shows what's currently playing.
queue • Shows the music that's currently queued.
restart • Restart the current song.
repeat • Set if the music player should repeat.
voteskip • Vote to skip the current music track.
volume • Set the volume of the music player.
jump • Set the time marker of the music playback.
radio • Stream random songs from radio stations.
youtube • Search and see YouTube results.
soundcloud • Search and see SoundCloud results.
bassboost • Boost the bass in Octave's audio.
cleanup • Cleanup the queue (duplicated, users who left)
filter • Apply audio filters to Octave's audio (such as speed and pitch)

_settings autorole

Set auto-roles that are assigned to users on joining.

Current auto-role: None

Arguments

set (@role) • Set the auto-role.

unset • Unset the auto-role.

Melmsie
_settings commands
Octave
_settings commmands

Enable or disable certain commands.

Arguments

enable (cmd) • Enable a command.

disable (cmd) • Disable a command.

list • List disabled commands.

Melmsie
_settings ignore
Octave
_settings ignore

Make the bot ignore certain users, channels or roles.

Arguments

user (user) • Ignore/unignore a user.

channel (channel) • Ignore/unignore a channel.

role (role) • Ignore/unignore users with a role.

list • List ignored entities.
